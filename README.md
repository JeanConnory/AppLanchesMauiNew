# 🍔 Lanchonete App / Snack Bar App

Aplicativo mobile feito em **.NET MAUI** que consome uma **Web API .NET 8**.  
O usuário pode explorar lanches (hambúrgueres, sucos, batata frita etc.), pesquisar, ver por categoria, adicionar ao carrinho e fechar pedidos.

Mobile app built with **.NET MAUI**, consuming a **.NET 8 Web API**.  
Users can browse snacks (burgers, juices, fries, etc.), search, view by category, add to cart, and place orders.

---

## 🚀 Funcionalidades / Features
- 🍔 Visualizar lanches / View snacks  
- 🔍 Pesquisar / Search  
- 📂 Categorias / Categories  
- 🛒 Carrinho / Cart  
- ✅ Finalizar pedido / Checkout  

---

## 🛠️ Tecnologias / Tech Stack
- .NET MAUI  
- .NET 8 Web API  
- Entity Framework Core  
- SQLite  

---

## ▶️ Execução / Run
```bash
git clone https://github.com/JeanConnory/AppLanchesMauiNew.git
cd LanchoneteApp.API
dotnet run
cd ../LanchoneteApp.Mobile
dotnet build
dotnet run
```
